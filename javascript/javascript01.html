01.변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        console.log(x);
        console.log(y);
        console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수는 데이터를 저장할 수도 있지만, 변경도 가능합니다. 

{
        let x = 100;

        let z = "javascript";

        x = 300;
        y =400;
        z = "jquery"

        console.log(x);
        console.log(y);
        console.log(z);
}

400

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

변수는 프로그래밍에서 데이터를 저장하고 변경하며, 필요에 따라 데이터를 추가할 수 있는 공간입니다. 

{
        let x = 100;

리터럴 방식으로 배열을 선언하면서 초깃값을 설정하면 JavaScript에서 간편하게 배열을 생성하고 초기화할 수 있습니다. 이를 통해 한 줄의 코드로 배열을 생성하고 값을 할당할 수 있습니다.

10. 배열 : 데이터 저장(여러개) : 표현방법6 : 다차원 배열

다차원 배열은 배열 안에 배열이 포함된 배열입니다. 이를 통해 행렬(matrix) 또는 다중 차원 데이터를 표현할 수 있습니다.

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열구조분해할당

배열 구조 분해 할당은 배열의 각 요소를 추출하여 변수에 할당하는 방법입니다. 이것은 ES6(ECMAScript 2015)에서 도입된 기능입니다. 

{
        const arr = [100, 200, "javascript"];

        const [a, b, c] = arr;

        console.log(a);
        console.log(b);
        console.log(c);
}
{
        const arr = [100, 200, [300, 400]];
        const [a, b, [x, y]] = arr;

        console.log(a);
        console.log(b);
        console.log(x);
        console.log(y);
}

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼심연산자

배열 펼침 연산자는 배열의 요소를 추출하여 새로운 배열이나 함수 호출 등에서 사용할 수 있도록 도와주는 문법입니다. 이 연산자는 ...으로 표현됩니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500];

        console.log(arr1);
        console.log(...arr1);        //...쓰면 쉼표가 사라짐
        console.log(...arr1, ...arr2);   
        console.log(...arr3);
}
결과 확인하기

13. 객체 : 데이터 저장(키와값) : 표현방법 1 : 객체 선언 후 배열 방식으로 데이터 입력

객체(Object)는 키(key)와 값(value) 쌍으로 데이터를 저장하는 자료구조입니다. 다양한 프로그래밍 언어에서 객체를 사용할 수 있으며, 이를 선언하고 데이터를 배열(array) 방식으로 입력하는 방법은 언어에 따라 조금씩 다를 수 있습니다. 

{
        const obj = new Object();   //객체 선언

        obj[0] = 100;   //obj 변수에 숫자 100을 저장
        obj[1] = 200;   //obj 변수에 숫자 200을 저장
        obj[2] = "javascript";   //obj 변수에 문자 javascript를 저장

        console.log(obj[0]);
        console.log(obj[1]);
        console.log(obj[2]);
}
결과 확인하기

20. 객체 : 데이터 저장(키와값) : 표현방법 8 : 객체 + 함수가 있는 방식

객체 내에 함수를 포함하는 방식은 메서드(method)를 정의하여 객체의 동작을 표현하는 방법입니다. 이를 통해 데이터와 해당 데이터를 조작하는 함수를 함께 저장할 수 있습니다. 

{
        const obj = {
            a: 100,
            b: [200, 300],
            c: "javascript",
            d: function () {
                console.log("javascript가 실행되었습니다.");
            },
            e: function () {
                console.log(obj.c + "가 실행되었습니다.");
            },
            f: function () {
                console.log(this.c + "가 실행되었습니다");
            }
        }

        console.log(obj.a);
        console.log(obj.b[0]);
        console.log(obj.b[1]);
        console.log(obj.c);
        obj.d();
        obj.e();
        obj.f();
}
