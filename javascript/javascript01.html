01.변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        console.log(x);
        console.log(y);
        console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수는 데이터를 저장할 수도 있지만, 변경도 가능합니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        x = 300;
        y =400;
        z = "jquery"

        console.log(x);
        console.log(y);
        console.log(z);
}
결과 확인하기
300
400
javascript

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

변수는 프로그래밍에서 데이터를 저장하고 변경하며, 필요에 따라 데이터를 추가할 수 있는 공간입니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        x += 300;
        y += 400;
        z += "jquery";

        console.log(x);
        console.log(y);
        console.log(z);
}
결과 확인하기
400
600
javascriptjquery

04. 상수 : 데이터 저장 + 데이터 변경(X)

상수는 프로그래밍에서 데이터를 저장할 수 있지만, 한 번 값이 할당되면 변경할 수 없는 공간입니다. 

{
        let x = 100;

        let z = "javascript";

        // x = 300;

        console.log(y);

}

05. 배열 : 데이터 저장(여러개) : 표현방법1 : 배열 선언

배열은 여러 개의 데이터를 저장하는데 사용되며, 선언할 때 대괄호 []를 사용합니다. 

{
        const arr = new Array();

        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);
}
결과 확인하기
100
200
javascript

06. 배열 : 데이터 저장(여러개) : 표현방법2 : 배열 선언과 동시에 초깃값 설정

배열은 여러 개의 데이터를 저장하는데 사용되며, 초깃값을 설정하려면 대괄호 안에 데이터를 나열하여 배열을 선언합니다. 

{
        const arr = new Array(100, 200, "javascript");

        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);
}
결과 확인하기
100
200
javascript

07. 배열 : 데이터 저장(여러개) : 표현방법3 : 리터럴 방식으로 배열 선언

배열은 여러 개의 데이터를 저장하는데 사용되며, 리터럴 방식으로 배열을 선언할 때는 대괄호 내부에 데이터를 나열하여 표현합니다. 

{

}

08. 배열 : 데이터 저장(여러개) : 표현방법4 : push() 메서드를 이용한 배열 방법

push() 메서드는 JavaScript에서 배열의 끝에 하나 이상의 요소를 추가하는 메서드입니다. 배열의 길이는 자동으로 조정됩니다. 

{
        const arr = [100, 200, "javascript"];

        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);
}
결과 확인하기
100
200

09. 배열 : 데이터 저장(여러개) : 표현방법5 : 리터럴 방식으로 배열 선언과 동시에 초깃값 설정

리터럴 방식으로 배열을 선언하면서 초깃값을 설정하면 JavaScript에서 간편하게 배열을 생성하고 초기화할 수 있습니다. 이를 통해 한 줄의 코드로 배열을 생성하고 값을 할당할 수 있습니다. 

{
        const arr = [100, 200, "javascript"];

        arr.push(400);

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);
        console.log(arr[3]);
}

10. 배열 : 데이터 저장(여러개) : 표현방법6 : 다차원 배열

다차원 배열은 배열 안에 배열이 포함된 배열입니다. 이를 통해 행렬(matrix) 또는 다중 차원 데이터를 표현할 수 있습니다. 

{
        const arr = [100, [200, 300, [400, 500, [600]]], ["javascript", "jquery"]];

        arr.push(400);

        console.log("10__________");
        console.log(arr[0]);
        console.log(arr[1][0]);
        console.log(arr[1][1]);
        console.log(arr[2][0]);
        console.log(arr[2][1]);
        console.log(arr[1][2][2][0]);
}

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열구조분해할당

배열 구조 분해 할당은 배열의 각 요소를 추출하여 변수에 할당하는 방법입니다. 이것은 ES6(ECMAScript 2015)에서 도입된 기능입니다. 

{
        const arr = [100, 200, "javascript"];

        const [a, b, c] = arr;

        console.log(a);
        console.log(b);
        console.log(c);
}
{
        const arr = [100, 200, [300, 400]];
        const [a, b, [x, y]] = arr;

        console.log(a);
        console.log(b);
        console.log(x);
        console.log(y);
}

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼심연산자

배열 펼침 연산자는 배열의 요소를 추출하여 새로운 배열이나 함수 호출 등에서 사용할 수 있도록 도와주는 문법입니다. 이 연산자는 ...으로 표현됩니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500];

        console.log(arr1);
        console.log(...arr1);
        console.log(...arr1, ...arr2);
        console.log(...arr3);
}
