03. 함수 : 매개변수 함수

    console.log(func1());

10. 화살표 함수 : 매개변수 + 리턴값

{

    }
    console.log(func1("10. 함수가 실행되었습니다."));

    console.log(func3("10. 함수가 실행되었습니다."));

    // 선언적 함수(가독성X)
    func4 = str => str;

    // 호출

    info2.result();
}

20. 함수 유형: 즉시실행 함수

함수를 정의함과 동시에 즉시 실행되는 JavaScript 함수입니다. 이러한 함수의 주요 목적은 함수 내에 있는 변수들을 전역 범위(global scope)로부터 격리시키고, 외부에서의 접근을 방지하여 변수명 충돌을 방지하고 코드를 모듈화하는 것입니다. 

{
    (function () {
        console.log("15. 실행되었습니다.");
    })();

    (() => {
        console.log("16. 실행되었습니다.");
    })();
}
결과 확인하기
15. 함수가 실행되었습니다. 16. 함수가 실행되었습니다.

21. 함수 유형: 파라미터 함수

다른 함수에 인자(argument)로 전달되는 함수를 가리킵니다. JavaScript에서 함수는 다른 함수의 인자로서 전달될 수 있으며, 이는 함수형 프로그래밍(Functional Programming)에서 매우 중요한 개념 중 하나입니다. 

{
    function func(str = "17. 실행되었습니다.") {
        console.log(str)
    }
    func();

    const func1 = (str = "18. 실행되었습니다.") => {
        console.log(str)
    }
    func1();
}
결과 확인하기
17. 함수가 실행되었습니다. 18. 함수가 실행되었습니다.
