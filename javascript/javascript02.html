05. 배열 : 데이터 불러오기 : for()문

배열에서 데이터를 불러와서 처리하기 위해 반복문을 사용하는 과정을 나타냅니다. 일반적으로 프로그래밍에서는 배열의 각 요소를 하나씩 접근하고 처리하기 위해 for 루프를 사용합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < num.length; i++) {
        console.log(num[i]);
    }

    for (let i = 0; i < num.length; i += 2) {
        console.log(num[i]);
    }
}

13. 객체 : 데이터 불러오기 : 기본

프로그래밍에서 객체를 사용하여 데이터를 불러오는 기본적인 방법을 나타냅니다. 객체는 키(key)와 값(value)의 쌍으로 이루어진 데이터 구조를 말하며, 키를 사용하여 해당 값에 접근할 수 있습니다.

}

14. 객체 : 데이터 불러오기 : Object.keys()

객체의 속성(키)을 배열로 반환하는 메서드입니다. 이 메서드는 주어진 객체의 모든 속성 이름을 배열로 반환하며, 이 배열은 해당 객체의 속성 순서와 동일한 순서로 속성 이름을 포함합니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(Object.keys(obj));
}
결과 확인하기
['a', 'b', 'c']

    console.log(Object.values(obj));

16. 객체 : 데이터 불러오기 : Object.entries()

객체의 키-값 쌍을 [키, 값] 형태의 배열로 반환하는 메서드입니다. 이 메서드는 주어진 객체의 모든 키-값 쌍을 배열로 반환하며, 반환된 배열은 배열의 요소로 각 키-값 쌍을 포함합니다. 

{
    const obj = {
        a: 100,

}

17. 객체 : 데이터 불러오기 : Object.assign()

하나 이상의 소스 객체로부터 대상 객체로 속성을 복사할 때 사용되는 메서드입니다. 이 메서드는 대상 객체를 첫 번째 인수로 받고, 그 다음 인수로 전달된 하나 이상의 소스 객체에서 대상 객체로 속성을 복사합니다. 속성 복사는 소스 객체의 순서대로 이루어지며, 소스 객체가 같은 키를 가지고 있는 경우 이전 속성은 덮어씌워집니다. 

{
    const obj1 = {
        a: 100,
        b: 200,
    }
    const obj2 = {
        c: 300,
        d: 400,
    }
    const obj3 = Object.assign(obj1, obj2);

    console.log(obj3);
}
결과 확인하기
{a: 100, b: 200, c: 300, d: 400}

18. 객체 : 데이터 불러오기 : hasOwnProperty()

객체가 특정 속성을 직접 소유하고 있는지 여부를 나타내는 메서드입니다. 이 메서드는 객체가 주어진 속성을 직접 소유하고 있는 경우에만 true를 반환하며, 프로토타입 체인 상에서 상속받은 속성은 고려하지 않습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    console.log(obj.hasOwnProperty("a")); 
    console.log(obj.hasOwnProperty("b")); 
    console.log(obj.hasOwnProperty("c")); 
    console.log(obj.hasOwnProperty("d"));
}
결과 확인하기
true
true
true
false

19. 객체 : 데이터 불러오기 : for in

객체의 속성을 열거하기 위한 반복문입니다. for...in 루프는 객체의 속성을 반복하면서 각 속성에 대해 특정 작업을 수행할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    for (let el in obj) {
    console.log(el + ":" + obj[el]);
}
}
결과 확인하기
a:100
b:200
c:javascript
