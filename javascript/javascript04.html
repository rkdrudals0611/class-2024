01. if문 (조건식)

if 문은 조건을 검사하여 참이면 특정 코드 블록을 실행하는 파이썬의 제어문이다. if,`else,elif 등의 키워드를 사용하여 다양한 조건에 따라 실행 흐름을 제어할 수 있다. 

{
    //true "문자열", 1, 2, "1", "2", [], {}
    //false  ""(빈문자열), 0, null, undefined

    if (0) {
        console.log("실행되었습니다.(true)");
    } else {
        console.log("실행되었습니다.(false)");
    }
}

실행되었습니다.(false)

02. 다중 if문

다중 `ifif문은if,`elif,`else 키워드를 사용하여 여러 조건을 처리하며, 각 조건은 이전 조건이 거짓일 때만 실행된다. 따라서 조건의 순서가 실행 결과에 영향을 미친다. 

{
    const num = 100;

    if (num == 90) {
        console.log("실행되었습니다.(90)");
    } else if (num == 100) {
        console.log("실행되었습니다.(100)");
    } else if (num == 110) {
        console.log("실행되었습니다.(110)");
    } else if (num == 120) {
        console.log("실행되었습니다.(120)");
    } else if (num == 130) {
        console.log("실행되었습니다.(130)");
    } else {
        console.log("실행되었습니다.");
    }
}
결과 확인하기
실행되었습니다.(100)

03. 중첩 if문

중첩 if 문은 하나의 if 문 안에 다른 if 문을 포함하여 복잡한 조건을 다루거나 여러 상황을 처리할 수 있도록 해주는 파이썬의 제어 구조이다. 이를 통해 더 세부적인 조건을 검사하고 다양한 상황을 처리할 수 있다. 그러나 코드의 가독성을 해치므로 필요할 때만 사용하는 것이 좋다. 

{
    const num = 100;
    if (num == 100) {
        console.log("실행되었습니다.1");
        if (num == 100) {
            console.log("실행되었습니다.2");
            if (num == 100) {
                console.log("실행되었습니다.3");
            }
        }
    } else {
        console.log("실행되었습니다.4")
    }
}
결과 확인하기
실행되었습니다.1
실행되었습니다.2
실행되었습니다.3

04. if문 생략 & 삼항 연산자

if 문을 생략하고 삼항 연산자를 사용하면 간단한 조건에 따라 값을 할당하거나 표현식을 평가할 수 있습니다. 이를 통해 코드를 간결하게 작성할 수 있으며, 가독성을 향상시킬 수 있습니다. 

{
    const num = 100;

    if (num == 100) {
        console.log("true");
    }

    if (num == 100) console.log("true")

    if (num == 100) {
        console.log("true");
    } else {
        console.log("false")
    }

    // 삼항 연산자(조건식 연산자)
    num == 100 ? console.log("true") : console.log("false");
}

05. switch문

switch 문은 프로그래밍에서 여러 조건에 따라 다른 동작을 수행하는 제어 구조이다. 파이썬은 switch 문을 직접 지원하지 않지만, 여러 개의 if나 elif 문을 사용하여 유사한 기능을 구현할 수 있다. 파이썬 3.10부터는 match 문이 추가되어 switch 문과 유사한 작업을 수행할 수 있다. 일반적으로 switch 문은 여러 개의 선택 사항을 간결하게 처리할 때 사용된다. 

{
    const num = 100;

    switch (num) {
        case 90:
            console.log("실행되었습니다.(90)");
            break;
        case 95:
            console.log("실행되었습니다.(95)");
            break;
        case 100:
            console.log("실행되었습니다.(100)");
            break;
        case 105:
            console.log("실행되었습니다.(105)");
            break;
        default:
            console.log("실행되었습니다.")
    }
}
결과 확인하기
실행되었습니다.(100)

06. while 문

while 문은 주어진 조건이 참일 때 반복적으로 코드를 실행하는 파이썬의 제어 구조이다. 조건이 참인 동안 코드 블록이 반복 실행되며, 조건이 거짓이 되면 반복이 종료된다. 이를 통해 반복 작업이 필요한 상황에서 코드를 간결하게 작성할 수 있다. 

{
    let num = 1;    //초깃값;

    while (num < 10) {  //조건식;
        console.log(num);   // 실행문
        num++;  //증감값;
    }
}

07. do while 문

do while 문은 조건을 먼저 확인하지 않고 일단 코드 블록을 실행한 후에 조건을 확인하여 반복을 계속할지 결정하는 제어 구조이다. 파이썬에는 기본적으로 do while 문이 없지만, 반복문을 사용하여 비슷한 동작을 구현할 수 있다. 이는 최소 한 번의 반복을 보장하고 조건을 반복문 내에서 확인한다. 

{
    let num = 1;  //초깃값;

    do {
        console.log(num);   //실행문
        num++;      // 증감값;
    } while (num < 10);     // 조건식
}

08. for문

for 문은 순회 가능한(iterable) 객체의 각 요소에 대해 반복적으로 코드를 실행하는 파이썬의 제어 구조이다. 주어진 iterable에서 요소를 차례로 꺼내어 코드 블록을 실행하며, 모든 요소를 처리할 때까지 반복한다. for 문은 리스트, 튜플, 문자열 등과 같은 다양한 데이터 구조를 처리하는 데 유용하다. 

{
    for (let i = 1; i < 10; i++) {       // 초깃값; 조건식; 증감값;
        console.log(i);
    }
}

09. 중첩 for문

중첩 for 문은 하나의 for 문 안에 또 다른 for 문을 포함하여 반복 작업을 수행하는 파이썬의 제어 구조이다. 외부 for 문의 각 반복마다 내부 for 문이 모두 실행되며, 모든 가능한 조합을 순회하면서 코드 블록이 실행된다. 이를 통해 다차원 배열이나 격자 형태의 데이터를 처리하는 데 유용하다. 

{
    for (let i = 1; i < 10; i++) {
        for (let j = 1; j < 10; j++) {
            console.log(i, j);
        }
    }
}

10. break

break는 반복문 내에서 사용되며, 해당 반복문을 즉시 종료하고 루프를 벗어나게 하는 제어문이다. 조건이 충족되는 경우에만 사용되며, 루프 내에서 break가 실행되면 반복문은 즉시 종료되고 해당 루프 다음의 코드가 실행된다. 이를 통해 특정 조건이 충족되면 반복 작업을 중단할 수 있다. 

{
    for (let i = 1; i < 10; i++) {
        if (i == 5) {
            break;
        }
        console.log(i)
    }
}

11. continue 문

continue문continue가 실행되면 현재 반복이 중단되고 다음 반복이 시작된다. 이를 통해 특정 조건을 만족하는 경우에만 반복 작업을 건너뛸 수 있다. 

{
    for (let i = 1; i < 10; i++) {
        if (i == 5) {
            continue;
        }
        console.log(i)
    }
}
